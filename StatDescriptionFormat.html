Stat Description Format

Description of the format of the file Bundles\Metadata\StatDescriptions\stat_descriptions.txt

Generalized Format

description
   NSTATS STAT1_KEY STAT2_KEY .. STATN_KEY
   NROWS
      STAT1_VALUE STAT2_VALUE .. STATN_VALUE STR [MUTATORS..]

Example: Single Stat

description
   1 avoid_interruption_while_casting_%
   2
      1|99 "{0}% chance to Avoid interruption from Stuns while Casting"
      100|# "Avoid interruption from Stuns while Casting"

Tips

The | symbol does not mean "OR"; the | symbol is used to define a range.

  • 1|99 means "match any numbers in the range 1-99"
  • 100|# means "match any numbers in the range 100 to positive infinity"
  • #|-1 means "match any numbers in the range negative infinity to negative one"
