  • ExperiencePenalty

  • DelimiterSearch
    • There is an 8 byte delimiter 0xBB_BB_BB_BB_BB_BB_BB_BB that marks the end of the record data and the beginning of the auxilliary data of each .dat64. There's little reason for it to exists, and it could coincidentally appear in other places of the file. This page lists the result of searching for it.
  • MutatorList
    • A count of every unique set of mutators for the file Bundles\Metadata\StatDescriptions\stat_descriptions.txt (english-only).
  • StatDescriptionList
    • Just Bundles\Metadata\StatDescriptions\stat_descriptions.txt formatted as an HTML table (english-only).
